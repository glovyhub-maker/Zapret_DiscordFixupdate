## Обход блокировки Discord и YouTube с помощью `zapret`

> **Рабочий метод для РФ, Беларуси, Украины и других стран, где применяется DPI-блокировка.**  
> Не требует VPN, прокси или дополнительных расширений. Просто, быстро, безопасно.

---

## :pushpin: Что это?

`zapret` – утилита для обхода **глубокого анализа пакетов (DPI)**, используемого провайдерами для блокировки Discord, YouTube, Telegram и других сервисов.  
Данный репозиторий содержит **готовые конфиги, скрипты и инструкции** для Windows и Linux, чтобы запустить `zapret` одной командой и забыть о блокировках.
